Privacy Policy

Last updated: June 20, 2026

This Privacy Policy explains how the Wordle Reminder Discord Bot ("the Bot", "the Service") handles information when used in a Discord server.

1. Information We Process

The Bot reads recent messages in a configured Discord channel to determine whether tracked users have posted a Wordle result for the current day. It also processes configured Discord user IDs and channel IDs required for reminder functionality.

2. How Information Is Used

Information is used only to:

  • Detect whether a daily Wordle result was posted.
  • Identify which tracked users should receive a reminder mention.
  • Send reminder messages to the configured channel.

3. Data Retention

The Bot does not persist message content in a database. Message content is processed in-memory during a Lambda invocation and then discarded.

4. Data Sharing

The Bot does not sell, rent, or share personal data with third parties. Data is only exchanged with Discord's API and AWS services required to run the bot.

5. Security

Reasonable technical measures are used to protect service credentials and runtime configuration. However, no system can be guaranteed 100% secure.

6. Your Choices

Server administrators can stop all processing by removing the Bot from their server or deleting its deployment.

7. Children's Privacy

The Service is not directed to children under the minimum age required by Discord and applicable law.

8. Changes to This Policy

This Privacy Policy may be updated from time to time. Changes are effective when posted on this page, and the "Last updated" date will be revised.

9. Contact
